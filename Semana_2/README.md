# Sistema de Agricultura
Se tiene un sistema de agricultura de precisión donde existen:

- **Estaciones Base**: Dispositivos que reciben datos de sensores y los transmiten a la nube.
- **Sensores de Suelo**: Dispositivos fijos que miden características del suelo como humedad, temperatura y pH.
- **Sensores de Cultivo**: Dispositivos fijos que evalúan el estado del cultivo a través de índices vegetales y estrés hídrico.

Cada sensor se conecta a **una estación base** y envía sus mediciones. El sistema debe poder:

1. Registrar estaciones base.
2. Crear sensores de suelo y cultivo.
3. Conectar cada sensor a una estación.
4. Realizar mediciones.
5. Mostrar información de sensores y estaciones.
